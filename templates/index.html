Tableau Superstore

Tableau vs. JavaScript

  • Data Source: Tableau Superstore
  • Github: Github


    • Our starting point for this project was a Tableau Superstore dataset. It has the advantages of being clean data, covering five years of operations, and it includes enough variables for numerous visualization possibilities.

    Initial data was in an Excel spreadsheet, which we uploaded into a PostgreSQL database using SQLAlchemy. This script, along with the data, was shared among the team members, allowing us to work independently while using common queries. A Flask app was coded in Python to provide RESTful APIs. These routes are accessed in JavaScript, to pull data from our queries, and are also used in our HTML to navigate between our home page, graphs, and raw data. The line and bar graphs are from the D3.js library, which offers Scalable Vector Graphics, and this was used to show profits by date and product category. Our US Chloropleth map is from the Plotly Graphing Library, and represents five years of profit and loss by state.

    The HTML index is built on a Bootstrap framework, containing a fluid jumbotron and Navbar with dropdown menu, all components being responsive for mobile compatibility.
